QuickDraw 模型训练 & CNN 卷积自学记录

2025.09.02 Niyao  热度 ℃

好问题 👍。我帮你把 QuickDraw 数据处理 → CNN 卷积提取特征 → Loss 计算 → Forward & Backward（反向传播） 全流程讲清楚，并配一张图解。

1. QuickDraw 数据处理流程

QuickDraw 原始数据是用户手绘的线条（stroke sequence），通常存储为 .npy 或 .npz 格式，里面是 28×28 灰度图。在训练 CNN 前，需要：

  1. 读取：加载 .npy 文件，得到 (N, 28, 28) 图像数组。
  2. 归一化：像素值从 [0,255] → [0,1]，方便梯度下降。
  3. 标签 one-hot：345 类别 → 转成 (345,) 的 one-hot 向量。
  4. 划分数据集：train / validation / test。

最终输入网络的就是： (batch_size, 28, 28, 1) 的张量。

2. CNN 卷积层工作原理

CNN 主要做两件事：

  • 卷积 (Convolution)：小卷积核（比如 3×3×输入通道）在图像上滑动，提取边缘/形状特征。
  • 池化 (Pooling)：缩小空间尺寸（比如 max pooling），保留主要特征。

例子：

  • 输入：28×28×1（灰度图）
  • 卷积核：3×3×1 → 输出 26×26×32（32 个特征图）
  • 再池化：13×13×32
  • 重复卷积 → 提取更抽象的形状特征（比如圆、直线）。

3. Loss 函数（交叉熵）

QuickDraw 是 多分类问题（345 类）。常用 Loss：

L=−∑i=1345yi⋅log⁡(y^i) L = -\sum_{i=1}^{345} y_i \cdot \log(\hat{y}_i) L=−i=1∑345​yi​⋅log(y^​i​)
  • yiy_iyi​：真实标签（one-hot，只有一个 1）
  • y^i\hat{y}_iy^​i​：softmax 输出的概率 目标：最大化正确类别的概率。

4. Forward Propagation（前向传播）

  1. 输入图像 → 卷积层（提取边缘特征）
  2. → 池化层（降维）
  3. → 更深卷积（提取复杂形状）
  4. → Flatten → 全连接层
  5. → Softmax 输出 345 维概率

5. Backward Propagation（反向传播）

训练时，模型计算 Loss，再进行梯度更新：

  1. 计算 Loss 对输出的梯度 ∂L∂y^\frac{\partial L}{\partial \hat{y}}∂y^​∂L​

  2. 反向传播到全连接层：更新权重 W, b

  3. 传播到卷积层：通过卷积核的梯度公式更新 filter 参数

  4. 传播到输入层（理论上，但一般不更新输入）

  5. 优化器（如 Adam / SGD） 用学习率 η 更新参数：

    W=W−η⋅∂L∂W W = W - \eta \cdot \frac{\partial L}{\partial W} W=W−η⋅∂W∂L​

6. 图解（流程示意）

这样一轮就是 一次 forward + backward，不断迭代直到 Loss 收敛。

1. 什么是归一化？

在深度学习里，归一化（Normalization） 指的是把数据调整到一个合适的数值范围。

最常见的几种：

  • Min-Max 归一化

    x′=x−min⁡(x)max⁡(x)−min⁡(x) x' = \frac{x - \min(x)}{\max(x) - \min(x)} x′=max(x)−min(x)x−min(x)​

    把数值缩放到 [0,1]。

  • 标准化 (Z-score Normalization)

    x′=x−μσ x' = \frac{x - \mu}{\sigma} x′=σx−μ​

    把数据调整到均值 0，方差 1。

在图像任务里，通常是：

  • 灰度像素 [0, 255] → [0, 1]
  • 或者 [0, 255] → [-1, 1]

2. 为什么要做归一化？

原因主要有 3 个：

(1) 避免数值过大，梯度爆炸 / 梯度消失

如果输入是 0~255，那么卷积时输出值会很大，softmax 里的 exp() 可能会溢出，导致 训练不稳定。 归一化后，数值变小，梯度更平稳。

(2) 提高训练速度，加快收敛

深度学习优化依赖梯度下降。 如果数据在同一量级（比如都在 [0,1]），网络更容易找到合适的更新方向，收敛更快。 否则，有些维度梯度很大，有些很小，优化会「抖动」。

(3) 提高准确率，避免模型偏向

例如 QuickDraw 的黑白画，有些人画得深（像素值大），有些人画得浅（像素值小）。 如果不归一化，模型会更关注「画深的人」，忽视「画浅的人」，导致泛化差。 归一化后，把不同画法映射到统一范围，模型更公平地学习特征。

QuickDraw 输入是 28×28 灰度图，像素范围 0~255。 常见做法：

这样图像就变成 0~1，利于 CNN 训练。

✅ 总结： 归一化就是把输入数据缩放/调整到更合适的数值范围，避免数值问题，提升收敛速度和模型准确率。

完全可以用这个公式来做像素归一化，我们仔细分析一下：

1. Min-Max 归一化公式回顾

  • xxx 是原始数值（例如像素值）
  • min⁡(x)\min(x)min(x) 是该数据集或该维度的最小值
  • max⁡(x)\max(x)max(x) 是该数据集或该维度的最大值
  • x′x'x′ 是归一化后的结果，范围一定在 [0,1][0,1][0,1]

2. QuickDraw 像素的范围

灰度图像每个像素的取值范围是：

x∈[0,255] x \in [0, 255] x∈[0,255]

所以：

min⁡(x)=0,max⁡(x)=255 \min(x) = 0, \quad \max(x) = 255 min(x)=0,max(x)=255

3. 套公式

把 min⁡(x)=0\min(x) = 0min(x)=0 和 max⁡(x)=255\max(x) = 255max(x)=255 代入公式：

x′=x−0255−0=x255 x' = \frac{x - 0}{255 - 0} = \frac{x}{255} x′=255−0x−0​=255x​

✅ 这样就把原本 0~255 的像素值，线性映射到了 [0,1]。

  • 原本 0 → 0
  • 原本 255 → 1
  • 中间值线性缩放，例如 128 → 128/255 ≈ 0.502

4. 为什么线性映射有效？

  1. 保留相对亮度：像素越亮，归一化后数值越大，卷积依然能感知明暗差异
  2. 便于梯度计算：CNN 处理小数值更稳定，避免梯度爆炸
  3. 统一尺度：不同手绘图像亮度不同，通过归一化映射到相同尺度，提高模型泛化

总结：

Min-Max 归一化公式是 线性映射，可以把任意数值范围 [min, max] 转换到 [0,1]，在 QuickDraw 中就是把灰度像素 [0,255] 缩放到 [0,1]，训练 CNN 更稳定、高效。

1️⃣ 单一模型 vs 多模型

QuickDraw 模型

  • 你现在训练的 QuickDraw CNN 模型：

    • 输出：345 个类别的概率
    • 单一任务单一模型，只做图像分类

  • 这个模型是 窄人工智能（Narrow AI）：只能做训练时定义好的任务，无法做其他类型任务。

GPT/Gemini 类多模态模型

  • 现代大模型，比如 GPT-4、Gemini、LLaMA-Adapter：

    • 输入可以是文本，也可以是图像，甚至一些模型在研究阶段支持视频、音频
    • 输出可以是文本生成、图像生成、回答问题、文本总结等

  • 架构上：

    • 通常是一个核心模型（一个 Transformer/大语言模型），但：

      • 对不同模态输入有 专门的嵌入层或编码器（image encoder、text tokenizer）
      • 这些编码器把不同模态的数据映射到同一个 通用向量空间

    • 输出阶段：

      • 再用 task-specific head（例如文本生成 head、图像生成 head）生成对应输出

  • 本质上是“一体多能”：底层是同一套大脑网络，只是不同类型输入经过不同的前置处理，然后共享底层理解能力。

2️⃣ 模型类比人脑

你说得很形象：

如果模型可以理解是深的大脑，对于人来说，理解外在事物的器官只有一个

  • 人脑有视觉皮层、听觉皮层、语言中枢等不同的“输入器官”，但底层计算和认知处理共享大脑神经网络

  • 类似地，AI 模型：

    • 图像、文本、语音输入各自经过不同的前置编码器（感官）
    • 编码后的向量进入共享 Transformer 或神经网络（认知核心）
    • 输出再分发到不同任务头（动作/表达）

所以现代多模态模型在架构上正在模拟人脑“单大脑多输入感官”的机制。

3️⃣ 算力对比人脑

  • 现代 AI 模型：

    • GPT-4（2023 年估计）参数在 数百亿到万亿级

    • 单纯参数量大，但：

      • 没有人脑的稀疏、高度并行动态调节能力
      • 记忆、即时学习能力远不如人脑（还要训练好几周才学完）

  • 结论：算力上接近人脑特定功能，但整体认知能力、常识推理、跨模态灵活性仍远落后于人类大脑。

1️⃣ CNN 卷积操作的直观理解

卷积在 CNN 中的作用本质是 提取局部特征。

假设输入是一个二维图像 XXX（大小 H×WH \times WH×W），卷积核 KKK（大小 kh×kwk_h \times k_wkh​×kw​）在图像上滑动，每次计算局部加权和，从而生成输出特征图 YYY。

直观公式：

Y(i,j)=∑m=0kh−1∑n=0kw−1K(m,n)⋅X(i+m,j+n) Y(i, j) = \sum_{m=0}^{k_h-1} \sum_{n=0}^{k_w-1} K(m, n) \cdot X(i+m, j+n) Y(i,j)=m=0∑kh​−1​n=0∑kw​−1​K(m,n)⋅X(i+m,j+n)
  • i,ji, ji,j 是输出特征图 YYY 的索引。
  • m,nm, nm,n 是卷积核内部索引。
  • 这就是 二维离散卷积（cross-correlation 形式）。

2️⃣ 卷积数学公式推导

卷积的连续定义（信号处理中的卷积）为：

(f∗g)(t)=∫−∞+∞f(τ)g(t−τ)dτ (f * g)(t) = \int_{-\infty}^{+\infty} f(\tau) g(t - \tau) d\tau (f∗g)(t)=∫−∞+∞​f(τ)g(t−τ)dτ
  • fff 是输入信号，ggg 是卷积核（滤波器）。
  • 离散图像中，积分变成求和：
(f∗g)[i,j]=∑m∑nf[m,n]⋅g[i−m,j−n] (f * g)[i,j] = \sum_m \sum_n f[m,n] \cdot g[i-m, j-n] (f∗g)[i,j]=m∑​n∑​f[m,n]⋅g[i−m,j−n]

在深度学习中，我们通常不做“翻转”，而直接用 cross-correlation：

Y[i,j]=∑m∑nK[m,n]⋅X[i+m,j+n] Y[i,j] = \sum_m \sum_n K[m,n] \cdot X[i+m, j+n] Y[i,j]=m∑​n∑​K[m,n]⋅X[i+m,j+n]

⚠ 注意：在数学上严格的卷积会翻转卷积核，但 CNN 框架（TensorFlow、PyTorch）直接用滑动窗口做加权和，更直观。

3️⃣ 加入偏置和非线性

CNN 不仅卷积，还会加上偏置 bbb 并经过激活函数 σ\sigmaσ：

Y(i,j)=σ(∑m=0kh−1∑n=0kw−1K(m,n)⋅X(i+m,j+n)+b) Y(i,j) = \sigma \Bigg( \sum_{m=0}^{k_h-1} \sum_{n=0}^{k_w-1} K(m,n) \cdot X(i+m,j+n) + b \Bigg) Y(i,j)=σ(m=0∑kh​−1​n=0∑kw​−1​K(m,n)⋅X(i+m,j+n)+b)
  • σ\sigmaσ 常用 ReLU、Sigmoid 或 Tanh。
  • bbb 允许模型学习偏移，增加表示能力。

4️⃣ 多通道卷积（RGB 图像）

对于多通道输入（如 RGB 图像 C=3C=3C=3），卷积公式扩展为：

Y(i,j)=∑c=0C−1∑m=0kh−1∑n=0kw−1Kc(m,n)⋅Xc(i+m,j+n)+b Y(i,j) = \sum_{c=0}^{C-1} \sum_{m=0}^{k_h-1} \sum_{n=0}^{k_w-1} K_c(m,n) \cdot X_c(i+m, j+n) + b Y(i,j)=c=0∑C−1​m=0∑kh​−1​n=0∑kw​−1​Kc​(m,n)⋅Xc​(i+m,j+n)+b
  • 每个卷积核对应输入的所有通道。
  • 可以学习组合不同通道的特征。
  • 输出通道数量 CoutC_{out}Cout​ 决定卷积核数量。

5️⃣ 卷积原理的推理与证明

卷积之所以能提取特征，可以从 线性代数和信号处理角度理解：

  1. 局部感受野 卷积核只关注局部像素，捕捉局部模式（如边缘、角点）。

  2. 平移不变性 卷积核在整个图像滑动，相同模式无论出现在图像哪个位置都能被检测到。

  3. 线性组合 + 非线性映射

    • 卷积 = 局部加权求和 → 相当于对输入空间做线性投影。
    • 激活函数引入非线性 → 可以表示复杂函数。

  4. 卷积的矩阵表示（向量化推理） 可以把卷积写成矩阵乘法：

    vec(Y)=W⋅vec(X) \text{vec}(Y) = W \cdot \text{vec}(X) vec(Y)=W⋅vec(X)
    • 其中 WWW 是稀疏矩阵，由卷积核重复填充构成。

卷积、积分、二重/多重级数的关系是什么

好的，我们来系统梳理 卷积、积分、二重/多重级数（求和）的关系，从连续到离散，再到高维，理清它们的数学联系。

1️⃣ 卷积与积分的关系（连续情况）

连续信号 f(t)f(t)f(t) 和 g(t)g(t)g(t) 的卷积定义是：

(f∗g)(t)=∫−∞+∞f(τ)g(t−τ) dτ (f * g)(t) = \int_{-\infty}^{+\infty} f(\tau) g(t - \tau) \, d\tau (f∗g)(t)=∫−∞+∞​f(τ)g(t−τ)dτ
  • 积分本质上是连续求和。
  • 卷积可以看作对输入信号 f(τ)f(\tau)f(τ) 进行 滑动加权平均，权重由 ggg 决定。
  • 积分起到“累加局部贡献”的作用。

🔹 直觉：把卷积理解为“把一个函数翻转、平移，然后做积分求和”，就是连续情况下的“滑动窗口加权求和”。

1.1 二维卷积（图像的积分形式）

对于二维连续信号 f(x,y),g(x,y)f(x, y), g(x, y)f(x,y),g(x,y)：

(f∗g)(x,y)=∫−∞+∞∫−∞+∞f(ξ,η) g(x−ξ,y−η) dξ dη (f * g)(x, y) = \int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(\xi, \eta) \, g(x-\xi, y-\eta) \, d\xi \, d\eta (f∗g)(x,y)=∫−∞+∞​∫−∞+∞​f(ξ,η)g(x−ξ,y−η)dξdη
  • 这是 二重积分，相当于把二维卷积核 ggg 在输入图像上滑动，每个位置做加权累积。
  • 如果再扩展到三维（例如视频或体数据），就变成 三重积分。

🔹 结论：卷积是积分的一种特殊形式，用于计算一个函数在另一个函数上的加权累积。

2️⃣ 离散卷积与级数（求和）的关系

离散信号 x[n],h[n]x[n], h[n]x[n],h[n] 的卷积定义：

y[n]=(x∗h)[n]=∑k=−∞∞x[k] h[n−k] y[n] = (x * h)[n] = \sum_{k=-\infty}^{\infty} x[k] \, h[n - k] y[n]=(x∗h)[n]=k=−∞∑∞​x[k]h[n−k]
  • 这里的求和就是 级数（series）。
  • 可以看作积分的离散化：积分 → 连续求和，求和 → 离散积分。

2.1 二维离散卷积

对于图像 X[i,j]X[i,j]X[i,j] 和卷积核 K[m,n]K[m,n]K[m,n]：

Y[i,j]=∑m∑nX[i+m,j+n] K[m,n] Y[i,j] = \sum_m \sum_n X[i+m, j+n] \, K[m,n] Y[i,j]=m∑​n∑​X[i+m,j+n]K[m,n]
  • 二重级数（double sum）：相当于二维积分的离散化。
  • 多通道或三维卷积则对应 三重或更高重级数。

  • f(τ)f(\tau)f(τ) → 离散输入数据（图像像素），没有简单解析式，只是一组离散数值。
  • ggg → 卷积核参数矩阵，通过训练学习得到，也没有固定解析式。
  • 输出 f∗gf*gf∗g → 离散加权和，通常是数值计算得到，不是闭式公式。

  • 公式直接写出来，例如高斯核、Sobel 算子等。

  1. 初始化卷积核 ggg（通常随机或 Xavier/He 初始化）。
  2. 输入 fff 经过卷积 + 激活 → 输出预测。

  4. 反向传播梯度 ∂L∂g\frac{\partial L}{\partial g}∂g∂L​ 更新卷积核权重。
  5. 训练多轮迭代后，卷积核 ggg 收敛 → 自动学会特征提取。

1️⃣ 卷积核不是手工设计的

在 CNN 里，卷积核 ggg 并不是你根据图像人工设计的（不像传统滤波器，例如 Sobel 边缘算子或高斯平滑）。

  • CNN 的优势是自动发现有用特征，无需人工指定。
  • 不同任务、不同数据集的卷积核可能完全不同。

换句话说，卷积核是数据驱动的，由训练过程自动优化。

  • Xavier/He 初始化：根据前后层神经元数量调整初始值，避免梯度消失或爆炸。

这个阶段，卷积核只是随机数矩阵，并没有特定含义。

  2. 特征图经过激活函数、池化等处理 → 最终得到预测值 y^\hat{y}y^​。

3.2 计算损失

  • 预测值 y^\hat{y}y^​ 与真实标签 yyy 计算损失函数 L(y^,y)L(\hat{y}, y)L(y^​,y)。
  • 例如分类任务用交叉熵损失，回归任务用均方误差。

3.3 反向传播（Backpropagation）

  • 对卷积核 ggg 计算梯度：

  • 梯度表示卷积核中每个元素应该如何调整才能让预测更准确。

g←g−η∂L∂g g \gets g - \eta \frac{\partial L}{\partial g} g←g−η∂g∂L​
  • η\etaη 是学习率。
  • 多轮迭代后，卷积核自动学会提取“对任务最有用的特征”。

4️⃣ 举例直观理解

假设你训练 CNN 做猫狗分类：

  • 第一层卷积核可能学会检测边缘、方向线条，类似 Sobel 滤波器。
  • 第二层卷积核可能学会检测局部纹理、耳朵或毛发模式。
  • 第三层卷积核可能学会组合前面特征，识别猫耳朵+眼睛+鼻子模式。

这里的卷积核不是实验设计的，而是通过大量图像训练得到的数值矩阵。

5️⃣ 总结

  1. 卷积核 ggg 初始阶段是随机数（或特殊初始化）。
  2. 训练过程中通过梯度下降优化，逐渐变成能提取任务相关特征的权重矩阵。
  3. CNN 卷积核不需要人工设计，完全依赖数据和损失函数来学习。
  4. 训练结束后，卷积核就是模型的“特征提取器”，数值上可以看作滤波器，但没有固定解析公式。
