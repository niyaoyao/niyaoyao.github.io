远程连接

用 ssh 远程连接 MySQL

如果出现 ERROR 2003 (HY000) 报错，则表明连接出现问题。

出现连接问题可能有三种原因：

  • 网络问题。
  • 端口问题。
  • 数据库配置问题。

网络问题

若能够 ping 通，则排除网络问题。

总结

GCP 比较坑的几个点是：

  • 必须将外部临时 IP 设置为静态 IP。
  • 必须配置防火墙规则以供外部访问相应端口。
  • 必须注释 MySQL 的配置 bind address，才能进行远程访问。
